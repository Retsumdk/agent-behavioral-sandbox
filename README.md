# agent-behavioral-sandbox

Isolated environment for testing and verifying agent behaviors before production deployment

## Features

- Production-ready code
- TypeScript/Python with full type safety
- Comprehensive error handling
- Built by Retsumdk

## Installation

```bash
git clone https://github.com/Retsumdk/agent-behavioral-sandbox.git
cd agent-behavioral-sandbox
bun install
```

## Usage

```bash
bun run src/index.ts --help
```

## Configuration

Create `config.json` (TypeScript) or `config.yaml` (Python) for custom settings.
